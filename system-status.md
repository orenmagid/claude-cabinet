Surfaces during `/verify learn` discover/draft when scanned components match a recipe; `discover.md` also probes routing shape (hash vs path). Depth-first principle is now load-bearing: a no-op stub for an un-exercisable step is worse than skipping that step entirely.
- Verify test-stack opt-in: `install.sh` emits an `e2e/start-test-stack.sh` launcher + `e2e/README.md` when `CABINET_VERIFY_TEST_STACK=1` at install time. Calibrate phase 2a-2d interview surfaces the scaffold only for projects that need an isolated test stack; off by default to keep cold-start fast.
